01. Project

  • App 이름

    : 식물도감

  • App 소개

    : 산림청 국립수목원에서 제공하는 식물의 정보를 찾아볼 수 있는 식물도감 어플입니다.

  • 선정 이유

    : 코로나 이후 식물에 대한 수요가 늘고 있는데 국내의 식물 관련한 대표적인 어플이 없으며, 해외 어플은 번역 문제가 있어, 식물 관련 어플에 개선이 필요하다고 생각하게 됐습니다.
    관련 어플을 찾던 중, 디자인이 단순했던 “식물도감” 어플을 선정하게 되었습니다.

02. 기존 App 문제점

  • 사용이 불편한 디자인

    → 검색을 할 때 자동완성 기능을 지원하지 않고 있으며, 찾을 때 가나다 순의 세로 스크롤을 내려야 식물을 볼 수 있었습니다.

  • 식물 검색 기능만 지원

    → 사진 검색이나 즐겨찾기 등의 다양한 기능을 지원하는 다른 어플에 비해 기능이 부족했습니다.

03. 진행과정

어플 선정 → 경쟁앱 선정,자료 조사 → 무드보드+ 임시 메인 화면 제작
→ UI/UX 방향 설계 → 정보구조도 작성 → 와이어프레임 작성 →
프로토타입 작성 → 최종완성 및 발표 → 완료보고서 작성

04. 담당업무

  • 자료 조사
  • 로고 제작
  • 일러스트레이터 작업
  • 찜콩 목록 (즐겨찾기 페이지) 디자인
  • 스타일 가이드 작성

05. UI/UX 방향

  • 검색 기능 다양화

    : 여러 검색 기능을 지원하여 편의성 증가를 목표했습니다.

  • 즐겨찾기 기능 추가

    : 즐겨찾기 기능을 추가해 나만의 식물도감을 가지고 싶어하는 사용자의 니즈를 충족시킬 수 있도록 했습니다.

  • 전 연령층에서 사용이 편리한 UI/UX

    : 처음 정보 수집 단게에서 주 연령층을 어디로 할 지 고민했습니다.
    여러 정보 기사와 네이버 검색 트렌드 등을 참고한 결과, 식물도감에 대한 검색량이 20~40대보다 50~70대가 더 높게 나왔으며 모든 연 령층이 비슷하게 집계되어 직관적인 UI 디자인을 가질 수 있도록 신 경썼습니다.
