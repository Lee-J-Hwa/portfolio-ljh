Team Project 목표 :

JavaScript를 활용한 웹사이트를 만들어보는 것이 목표였습니다.
팀 의견에 따라 canon 공식 사이트를 리디자인 하게 되었습니다.

Style Guide

Logo

캐논 회사의 로고입니다.
흰색이 아닌 버전의 로고도 있었지만,
작업한 메인 사이트엔 사용되지 않았습니다.

Color

검은색을 메인컬러로 깔끔하고 세련된 느낌으로
잘 디자인 된 거 같습니다

Font

영문은 NeueMontreal 폰트를 사용하고,
한글은 Noto sans KR 폰트를 사용했습니다.

Skill

HTML5＆CSS3

웹표준,접근성을 고려한 시멘틱 마크업
flex＆grid 태그를 사용한 레이아웃
@keyframes와 animation을 이용한 효과 구현

JavaScript

로고의 크기 변화하는 스크롤 이벤트
스와이퍼 라이브러리를 활용한 슬라이드 구현

문제 및 해결 :

＊로고 이미지 깨짐 문제

→ 스크롤 이벤트로 로고 크기가 늘어나고 줄어들면서 이미지가 깨지는 문제가 생겼고, SVG 파일로 교체해서 해결했습니다.

Project review

이번 작업은 자바스크립트가 많이 들어가야 했습니다.
부족한 자바스크립트 이해로 처음 구상했던 사이트와는
다른 모습으로 완성되어 아쉬움이 남지만,
자바스크립트를 배울 수 있는 좋은 기회가 되었습니다.
