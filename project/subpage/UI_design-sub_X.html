Style Guide

Logo

두꺼우면서 잘 보일 수 있는
UNPEOPLE Gothic UNI 폰트를
사용하였습니다.

Color

포인트 컬러 선정에서 남색과 초록색 중 고민하였습니다.
초록색이 식물 앱에 잘 어울려서
#03C75A 색상을 최종 선택하였습니다.

Font

제목과 소제목은 Montserrat 폰트를 사용하고
내용은 깔끔한 Noto sans 폰트를 사용하였습니다.

Skill

Graphics

피그마 사용하여 로고 제작
사용 설명 부분의 이미지 일러스트레이터로 작업
포토샵 활용하여 이미지 편집

문제 및 해결 :

＊버튼의 색 대비

→ 색 대비가 맞지 않아 사용자의 눈에 잘 안 보일 수 있는 문제가 생겼습니다.
텍스트 색을 흰색에서 검은색으로 변경하여 해결하였습니다.

Project review

팀 프로젝트를 시작할 때 디자인에 대해 잘 몰라서
팀에 도움을 많이 못 주었지만, 팀원의 작업하는 방법과
정보 검색을 통해 알아보며, 디자인 하는 방법에 대해
알 수 있게된 시간이었습니다.
