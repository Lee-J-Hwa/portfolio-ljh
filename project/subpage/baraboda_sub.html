01. Project

반응형 웹 사이트가 아니였던 바라보다 카페 공식 사이트를 리디자인 하였습니다.

03. Publishing

  • MediaQueries 태그 활용한 반응형 웹

    - Mobile First

    - Tablet : @media ( width>=1024px )

    - PC : @media ( width>=1440px )

  • flex, grid 를 이용한 웹페이지 레이아웃 제작
  • aos 플러그인을 활용한 Scroll event 구현

04. Problem &
Solution

  • 이미지 깨짐 현상 발생

    → 저해상도의 이미지 문제였으며, 높은 해상도의 이미지로 교체해 해결하였습니다.

  • 이미지 안의 텍스트 넘침

    → 장문의 텍스트를 이미지 안에 넣어 생긴 문제로 텍스트를 줄이는 방법으로 해결하였습니다.

05. Review

처음으로 만드는 반응형 웹이라, 레이아웃을 맞추는데 시간이 걸렸지만, 간단한 구조로 디자인 해서 어렵진 않았습니다. 스크롤 이벤트가 꼭 필요했는데, 찾아보던 중 aos라이브러리를 알게 되어 마지막에 추가하게 됐고 만족스럽게 작업을 마쳤습니다.
