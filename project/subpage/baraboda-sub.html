Project 목표 :

모바일과 태블릿 환경에 맞게 보여질 수 있는 반응형 웹 제작이 목표였습니다.
반응형 웹 사이트가 아니였던 바라보다 카페 공식 사이트를 리디자인 하였습니다.

Style Guide

Logo

바라보다 커피의 로고입니다.
상단,하단에 들어가는 2개 컬러로 나눠져 있으며,
로고의 색상 덕분에 포인트 컬러를 쉽게 정할 수 있었습니다.

Color

주조색,보조색을 무채색을 사용해 깔끔하게 만들었습니다.
포인트 컬러는 바라보다 로고 색상을 사용해 심심한 느낌을 지우고,
브랜드 이미지를 강조해주고 싶었습니다.

Font

폰트는 많이 사용되고 무난한
Pretendard 와 Noto sans 서체를
사용하였습니다.

Skill

HTML5＆CSS3

웹표준,접근성을 고려한 시멘틱 마크업
@Media-query 태그 사용 (반응형 웹)
flex＆grid 태그를 사용한 레이아웃

JavaScript

JS 라이브러리를 사용한 Scroll Event 구현

문제 및 해결 :

＊이미지 깨짐 현상

→ 저해상도의 이미지 문제였으며, 높은 해상도의 이미지로 교체해 해결하였습니다.

＊이미지 안의 텍스트 넘침

→ 장문의 텍스트를 이미지 안에 넣어서 생긴 문제로 텍스트를 줄이는 방법으로 해결하였습니다.

Project review

마지막에 추가하게 됐고 만족스럽게 작업을 마쳤습니다.
